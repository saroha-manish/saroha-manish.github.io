Landis+Gyr – Data Scientist

Project Overview

  • Poor visibility into user experience and delayed detection of network issues were affecting overall service quality and user satisfaction.
  • Developed a predictive system to monitor Quality of Experience (QoE) using MOS (Mean Opinion Score) estimation for proactive network management and issue detection.
  • Built a machine learning pipeline to predict MOS with 94% accuracy using historical network performance data.
  • Optimized multiple regression models—including Random Forest, XGBoost, and LSTM—to improve prediction robustness under varying conditions.
  • Integrated real-time MOS predictions into operational dashboards, enabling real-time tracking and early detection of service degradation.
  • Reduced customer complaints by 22% through proactive alerts and faster incident response, and significantly improved SLA adherence and user satisfaction metrics.
