Landis+Gyr

Role: Data Scientist

Project Overview

  • Service quality was impacted due to poor visibility into user experience and delayed detection of network issues.
  • A predictive solution was required to proactively monitor and address Quality of Experience (QoE) degradation.

Key Features

  • Developed a predictive system using Mean Opinion Score (MOS) estimation for monitoring telecom network QoE in real time.
  • Built machine learning models (Random Forest, XGBoost, LSTM) achieving 94% accuracy in predicting network quality scores.
  • Integrated model outputs into live dashboards for continuous monitoring and alerting of service degradation.
  • Enabled early warnings that reduced customer complaints by 22% and improved SLA adherence significantly.

Technology & Tools

  • Python, Pandas, NumPy, Scikit-learn, XGBoost, TensorFlow/Keras (LSTM)
  • Power BI / Tableau for real-time dashboards
  • Regression modeling, time series forecasting, data pipeline integration
