MOS PREDICTION

Project Overview

Developed a predictive system at Landis+Gyr to estimate Quality of Experience (QoE) before user calls using Mean Opinion Score (MOS) prediction. Leveraged machine learning models (Random Forest, XGBoost, LSTM) to achieve 94% accuracy, enabling proactive network adjustments. This approach reduced customer complaints by 22% and improved SLA compliance through early detection of potential service issues.

Key Features

  • Pre-Call MOS Prediction:: Estimated Mean Opinion Score (MOS) before user calls to enable proactive quality management.
  • Accurate Machine Learning Models: :Achieved 94% prediction accuracy using optimized Random Forest, XGBoost, and LSTM models.
  • Proactive Network Issue Detection:: Enabled early identification of potential service degradation to support preventive actions.
  • Operational Integration::Incorporated predictions into monitoring tools to guide network adjustments and reduce user-impacting issues.

Problem Statement

Limited visibility into network performance and delayed detection of quality issues were leading to poor user experience and increased customer complaints. There was a critical need for a proactive system that could predict the Quality of Experience (QoE), specifically the Mean Opinion Score (MOS), before user interactions occurred, enabling timely network adjustments and improving overall service reliability.

Solution Approach

Collected and processed historical network and QoE data to extract key features impacting call quality. Trained and optimized ML models (Random Forest, XGBoost, LSTM) to predict MOS with 94% accuracy. Integrated the model into monitoring workflows to estimate MOS before user calls. Enabled proactive network adjustments, reducing service issues and improving SLA compliance.

Results

  • 22% Reduction in Customer Complaints: Early detection of potential QoE degradation led to timely interventions.
  • Improved SLA Adherence:Proactive network adjustments ensured better compliance with service level agreements.
  • Enhanced User Satisfaction: Predictive alerts minimized service disruptions, boosting overall user experience.
  • Operational Efficiency:Streamlined network management through automated MOS predictions, improving response times and reducing manual oversight.

Challenges

Challenges included limited high-quality data for accurate MOS prediction and the complexity of multiple network factors impacting QoE. Ensuring model robustness across varying conditions and integrating predictions into existing systems posed technical hurdles. Gaining stakeholder confidence in a proactive system was also a key challenge. These factors required careful data processing and model optimization to overcome.

Future Work

Future work will focus on enhancing the model’s real-time capabilities and integrating more diverse data sources to improve accuracy. We aim to incorporate advanced network analytics and explore deep learning models for further optimization, while expanding the system’s scope to cover a broader range of services and devices.
