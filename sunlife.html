Agent Attrition Prediction Model

Project Overview

This project converts early agent activity data into actionable performance insights. Using the first three months of agent performance, we predict whether a rookie agent is likely to become a High Performing Agent (AC ≥ 15,000 per month). The system evaluates early sales momentum and provides probability scores to enable proactive coaching, structured onboarding, and better workforce planning decisions.

Key Features

  • Cohort-Based Modeling: Agents grouped by tenure to ensure fair performance comparison.
  • Early Momentum Detection: Uses AC, AFYP, policy count, and proposal activity from Month 1–3.
  • Multi-Model Evaluation: Compared tree-based models and AutoML to select the most stable performer.
  • Actionable Outputs: Generates probability scores and High Performer classification.
  • Explainability: Identifies key early indicators influencing future success.
  • Operational Integration: Outputs consumed via dashboards for manager-level monitoring.

Technologies and Tools

  • Programming: Python
  • Modeling: Random Forest, XGBoost, LightGBM, CatBoost, AutoML (Neural Network)
  • Data Processing: Pandas, NumPy
  • Visualization: Power BI dashboards
  • Storage: AWS S3 (processed data zone)

Problem Statement

In insurance sales organizations, performance assessment often happens too late. Managers lack early signals to identify high-potential agents or intervene with underperforming ones. We needed a reliable early prediction system that uses initial performance data to identify future high performers and support data-driven coaching strategies.

Solution Approach

Agents were structured into quarterly tenure-based cohorts. For each agent, performance metrics from the first three months were captured and used to train classification models predicting high performance outcomes. Time-based validation ensured realistic performance measurement.

  • Feature Engineering: Monthly AC, AFYP, policies, and proposal counts.
  • Model Training: Supervised classification with time-based validation.
  • Threshold Optimization: Balanced precision and recall for practical deployment.
  • Deployment: Risk scores surfaced to dashboards for managerial action.

Results

  • Strong Predictive Performance: Maintained stable accuracy and ROC-AUC on unseen cohorts.
  • Early Differentiation: Successfully identified high-potential agents within 3 months.
  • Balanced Precision & Recall: Optimized for practical business deployment.

Note: Metrics may vary based on cohort composition and deployment environment.

Challenges

Addressed class imbalance, ensured strict time-based validation to prevent data leakage, managed cohort alignment issues, and balanced model complexity with explainability for leadership stakeholders.

Future Enhancements

  • Behavioral Features: Incorporate engagement and manager-level signals.
  • Cost-Aware Optimization: Align thresholds with revenue and coaching capacity.
  • Automated Scoring: Real-time evaluation for new agent onboarding.
  • Workflow Integration: Direct linkage with coaching and performance management systems.
